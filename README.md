# Orders

## Project Setup
1. git clone https://github.com/AmanMinhas/Orders.git
2. cd Orders
3. npm i && npm start
4. In new terminal or new tab | cd client
5. npm i && npm start
6. Open project @ localhost:3000/

## Info
Using create-react-app and express-generator to bootstrap the project.
Using redux to save state on client, redux saga for async api requests.
